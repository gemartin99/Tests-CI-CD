# Uso 

Para ejecutar el tester clonar este repositorio y mover el archivo moulitest al repositorio del proyecto.

Ejecutar:
```gcc -Wall -Wextra -Werror -fPIC -shared starter.c -o libalumno.so```

Ejecutar el tester sobre la librería:
```./moulitest ./libalumno.so```

Si ✅ Todas las pruebas pasaron correctamente significa que la función se ha realizado correctamente.
